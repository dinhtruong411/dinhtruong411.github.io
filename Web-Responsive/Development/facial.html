美肌再生プログラム

効果：　シミ・シワ・たるみ・ニキビといった全ての肌トラブルの改善

施術：　クレンジング　→　ウォッシング　→　肌診断　→　ハンドマッサージ　→　イオン導入　→　セルサー　→　Intensiveローション塗布　→　　MaskRichesse塗布　→　仕上げ

定価:　￥30,000

※主にホームケア化粧品にてお肌のトラブルを改善していく美肌再生法です。

毛穴解消トリートメント

効果：　毛穴のつまり、ひきしめ、美白

施術：　クレンジング　→　洗顔　→　スクライバー　→　吸引　→　パック　→　仕上げ

毛穴につまった角栓皮脂の汚れを吸引器で吸引！洗顔では落としきれなかった汚れを取り除くトリートメント。

定価:1回　￥8,500

速攻美白ケア

効果：　美白、くすみ、キメ、シミ

施術：　クレンジング　→　洗顔　→　エレクトロクレンジング　→　イオン導入　→　仕上げ

チョ〜痛気持ちいい！から。筋肉運動、ツボ押し、そしてリンパの3つの要素を施術する「顔筋マッサージ」。新陳代謝を活発にしてゆきます。

定価:1回　￥13,000

生コラエステ

効果：　保湿、リフトアップ、シワ、ツヤ、ハリ、美白、小顔

施術：　クレンジング　→　洗顔　→　パッティングリフト　→　セルサー　→　パック　→　生コラーゲン塗布　→　仕上げ

コラーゲンと聞けば「ハリ」「ツヤ」「潤い」はもう常識、feminineによる生コラエステのコラーゲンは通常のコラーゲンより3〜6倍の保湿効果を持ち、瞬時にモチモチ肌get。

定価:1回　￥9,000

顔上げ矯正マッサージケア

効果：　小顔リフトUPに最適！むくみ、たるみ、シワ、リラクゼーション

施術：　クレンジング→洗顔→ヘッド・ネック・デコルテマッサージ　→　（耳ほぐし+美骨マッサージ）or（美筋マッサージ）　→　セルサー　→　パック　→　生コラ　→　仕上げ

「骨」or「筋肉」にアプローチすることにより「スッキリ、くっきり」とした引き締まった顔立ちと透明感のある艶やかな肌を持続させます。

定価:1回　￥11,000

デットクスマスク（オプション）

効果：　ニキビ、リフトアップ、たるみ、引締め

石膏マスクでマイナスイオン効果により皮脂や角栓等を排出しやすくなるのでニキビや吹き出物にも効果的。その他リフティング効果やたるみ改善、引締め効果も得られます。

定価:1回　￥3,000
